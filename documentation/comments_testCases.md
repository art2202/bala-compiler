# Bala Compiler Language Documentation 

## Glossary

| Reserved Word | Common Use | Obs |
|--- |--- |--- |
| caracter | char | --- |
| boleano | bool | --- |
| // | // | line comment |
| <\!--""--> | /\*\*/ | multi-line comment |


### Examples of comments

```cpp
// Correct

caracter c;
c = 'G';

c--;

<!--"
boleano d;
d = verdadeiro;

d--;
mostrar(d);  0 == falso "-->

mostrar(c); // F

```
