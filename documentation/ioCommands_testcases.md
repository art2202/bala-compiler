# Bala Compiler Language Documentation 

## Glossary

| Reserved Word | Common Use | Obs |
|--- |--- |--- |
| texto | string | --- |
| ler | cout | --- |
| mostrar | cin.getline | --- |


### Examples of IO Commands

```cpp
// Correct

texto teste;
ler(teste);

```


```cpp
// Correct

texto nome;
nome = "gabi";
mostrar(nome);

```
